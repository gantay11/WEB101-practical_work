
##  Documentation

In this project, I applied several important concepts related to web development and RESTful APIs:

- **REST API Methods**
  - GET → Fetch weather data
  - POST → Save location
  - PUT → Update location
  - DELETE → Remove location

- **API Integration**
  - Used OpenWeatherMap API to retrieve real-time weather data
  - Used JSONPlaceholder API to simulate backend operations

- **JavaScript Concepts**
  - Event handling
  - Fetch API for asynchronous requests
  - DOM manipulation for displaying data

- **User Interface Design**
  - Created forms and buttons for interaction
  - Displayed API responses dynamically
