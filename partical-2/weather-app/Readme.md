# RESTful API Weather Application

## Project Description
This project is a web-based application that demonstrates the use of RESTful APIs by implementing HTTP methods such as GET, POST, PUT, and DELETE.

It uses:
- OpenWeatherMap API (for fetching weather data)
- JSONPlaceholder API (for simulating database operations)
